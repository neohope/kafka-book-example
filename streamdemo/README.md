About
====================
Kafka strems demo

This is a code example that complements the material in the Kafka O'Reilly book. 


Requirements
====================
Requires Java 8 and Maven to run.


Build
====================
1. check out
2. mvn clean install


Run
====================
1. wordcount: 单词出现频率计算
2. avg: 计算平均价格
3. clickstreamenrich: 网站点击统计
4. stockstats: 股票价格统计
